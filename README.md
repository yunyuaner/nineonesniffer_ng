# 91Porn Site sniffer
Download videos from 91Porn

<b>Usage:</b>
<p>
<b>Download video descriptor</b></br>
&nbsp;&nbsp;&nbsp;&nbsp;sniffer -mode dl_desc -url video_page_url [-presist]
</p>
<p>
<b>Download video files using per-downloaded video descriptors</b></br>
&nbsp;&nbsp;&nbsp;&nbsp;sniffer -mode dl_video [-url video_page_url] [-transcode] [-persist]
</p>
<p>
<b>Sync the lastest video list ( prefetch + parse )</b></br>
&nbsp;&nbsp;&nbsp;&nbsp;sniffer -mode sync -count num [-proxy] [-keep]
</p>
<p>
<b>Identify video uploaded date</b></br>
&nbsp;&nbsp;&nbsp;&nbsp;sniffer -mode identify_date
</p>
